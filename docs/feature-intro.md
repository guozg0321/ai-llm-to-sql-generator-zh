# 功能详细介绍

## 一、智能查询功能
### 1.1 自然语言转SQL
- 支持中文口语化提问，自动识别查询意图
- 支持复杂查询：多表关联、分组统计、条件过滤、排序、分页
- 支持上下文对话，可基于上一次查询结果继续提问
- 实时显示生成的SQL语句，支持手动修改调整

### 1.2 智能纠错和推荐
- 自动识别错别字和表述不清的问题，给出修正建议
- 基于历史查询记录，智能推荐相关查询问题
- 支持常用查询保存为模板，一键复用

## 二、可视化功能
### 2.1 自动可视化
- 自动识别数据类型，选择最合适的图表类型
- 支持折线图、柱状图、饼图、散点图、仪表盘、雷达图、表格等多种图表
- 自动生成图表标题、坐标轴标签、图例说明

### 2.2 自定义仪表盘
- 拖拽式布局，自由组合多个图表
- 支持设置自动刷新频率，实时监控数据
- 支持仪表盘共享给团队成员

## 三、数据源管理
### 3.1 多数据源支持
- 关系型数据库：MySQL、PostgreSQL、Oracle、SQL Server、ClickHouse
- 非关系型数据库：MongoDB、Elasticsearch
- 文件类型：Excel、CSV、JSON
- 大数据平台：Hive、Spark SQL

### 3.2 数据安全
- 数据源连接信息加密存储
- 支持数据脱敏，敏感字段自动隐藏
- 操作审计日志，记录所有用户查询行为

## 四、团队协作功能
### 4.1 权限管理
- 基于角色的权限控制（RBAC）
- 支持数据源权限、查询权限、仪表盘权限细粒度管控
- 支持部门和用户组管理

### 4.2 协作功能
- 查询历史共享
- 仪表盘协作编辑
- 评论和@提醒功能
- 定时报表推送
